# Praxis UI Core

Biblioteca Angular que interpreta metadados provenientes do backend e gera telas dinamicamente.

```
frontend-libs/
└── praxis-ui-workspace/
    └── projects/
        └── praxis-ui-core/
```

## Desenvolvimento

1. Instale as dependências na raiz do workspace:
   ```bash
   npm install
   ```
2. Gere a biblioteca:
   ```bash
   npm run build praxis-ui-core
   ```
